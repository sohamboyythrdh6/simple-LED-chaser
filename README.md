# simple-LED-chaser
This project is a simple LED chaser circuit using the NE555 timer IC and CD4017 decade counter. The LEDs light up sequentially, creating a running light effect.
And I made this project to learn PCB design and understand how timing circuits and counters work together. It helped me practice using KiCad and designing a real circuit that can be manufactured.
